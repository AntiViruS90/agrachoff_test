Итак, сайт сделан согласно макету https://www.figma.com/file/UH1uQS1Q04DAUKVMRAcVr0/Untitled?node-id=0%3A1
	1. К иконке AGRACHOFF прикреплена ссылка перехода к главной странице
	2. При кликании на "МЕНЮ" появляется dropdown_menu, повтороное нажатие - меню исчезает, анимацию медленную настроил
	3. При нажатии на "ПОРТФОЛИО" переходим на страницу с формой, так как ранее была настроена для получения данных прогноза погоды, то не стал убирать, пусть остаётся. Форма подключена по API к сайту: https://api.openweathermap.org/ и теперь, при вводе названия города, выдаётся информация по погоде.

	5. В footer Отправить сообщение по email, нужно нажать на кнопку и форма появится, согласно полученной обратной связи, что email форма повлялась сама по себе. Протестировал на четырёх браузерах, форма появляется после нажатия кнопки "Отправить сообщение по email"
	6. Для скачивания файла c GitHub нажмите в меню "ОТЗЫВЫ"
	7. Форма для получения данных по API с сайта https://api.forum-auto.ru/docs/listgoods.php согласно поиску по артиклу находится в меню "ГАРАНТИЯ"
